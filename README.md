# rasterizer
Implementation of a software rasterizer

 
![](./prepective.png)
![](./orthographic.png)
![](./cow.png)
![](./square.png)


### Controls:

1. `UP Arrow`: Scale up

2. `Down Arrow`:Scale down

3. `Left Arrow`: Rotate around y clock-wise

4. `Right Arrow`:Rotate around y Counter-clock-wise
